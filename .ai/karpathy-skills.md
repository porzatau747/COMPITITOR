# Karpathy Skills for Coding Agents

## 1. Think Before Coding
State assumptions, uncertainty, and tradeoffs before editing.

## 2. Simplicity First
Implement the smallest solution that solves the task. Do not over-engineer.

## 3. Surgical Changes
Touch only files required by the task. Do not refactor unrelated code.

## 4. Goal-Driven Execution
Define success criteria, then verify with build/lint/tests when available.
